开发环境搭建

本书开发环境采用 Vagrant + Ubuntu14 搭建。

Vagrant：程序员的虚拟机

Vagrant是一款用ruby开发的控制虚拟机的工具，其可以很控制现有大部分的虚拟机，包括：VirtualBox, Vmware Workstation, KVM, Xen, 支持各种操作系统，包括：Linux、Windows、Mac。

开发环境搭建

git clone https://github.com/blockchain-university/devenv
cd devenv
vagrant up # 根据网络状况，需要下载时间比较长，请耐心等待
